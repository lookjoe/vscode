
# flutter使用GetX微框架时快捷生成文件

## 使用方法
  1.command+shift+p 搜索 Flutter: GetX Create Page 输入文件名称

  2.右键目录选择Flutter: GetX Create Page输入文件名称

生成文件目录
```
  demo_page
    logic.dart
    state.dart
    view.dart
```
